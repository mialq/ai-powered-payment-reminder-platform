# Política de Comunicação

## Objetivo

Definir regras mínimas para o envio de lembretes de pagamento em uma solução de priorização de clientes com foco em comunicação preventiva.

## Princípios Gerais

- A comunicação deve ter caráter preventivo, claro e profissional.
- O objetivo principal é lembrar o cliente sobre um pagamento próximo do vencimento, evitando atraso.
- A mensagem não deve ter tom agressivo, ameaçador ou de cobrança pesada antes do vencimento.
- A comunicação deve respeitar a elegibilidade do cliente para automação.

## Regras para Envio Automático

Um cliente pode seguir para automação quando atender simultaneamente aos critérios abaixo:

- possuir cadastro válido para contato
- não estar bloqueado por regra operacional
- não possuir risco desconhecido
- estar marcado como elegível para envio

## Regras para Bloqueio da Automação

A automação deve ser bloqueada quando ocorrer qualquer uma das situações abaixo:

- cliente sem cadastro válido
- cliente não elegível para envio
- cliente com risco desconhecido
- caso com necessidade explícita de revisão humana

## Diretrizes de Comunicação

- A mensagem deve ser curta e objetiva.
- O texto deve informar o vencimento próximo sem gerar atrito desnecessário.
- Sempre que possível, a mensagem deve permitir que o cliente desconsidere o aviso caso já tenha efetuado o pagamento.
- Em casos bloqueados, não deve ser gerada mensagem automática de cobrança.

## Resultado Esperado

A política de comunicação deve garantir que apenas clientes aptos sejam acionados automaticamente e que casos sensíveis sejam encaminhados para revisão manual.
